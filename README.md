# bolonggram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request secret
- [x] Confirm secret(Log in)
- [x] Like / Unlike a photo
- [x] Comment a photo
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow user
- [x] See user profile
- [x] See my profile
- [x] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit a photo (Delete)
- [ ] See the feed
